# Text File Formatter
Python script to format .TXT files.

## Description
Whenever I used to export my Kindle book notes, the formatting would be grammatically incorrect. Hence, the primary purpose for my writing this code was to speed up the time it took me to format my Kindle book notes, [which I share on my website](https://www.ravinderdeol.com/notes).
